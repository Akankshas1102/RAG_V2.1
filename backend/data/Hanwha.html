Overview

The scope of this document is to provide instructions on configuring a Hanwha NVR in Viewscape.

Pre-requisites

  • Viewscape version 8.36+

Installation

The basic steps for setting up a Hanwha NVR are:

  1. Create an NVR in Viewscape as normal
  2. Set NVR type to Hanwha
  3. Input NVR details as normal
  4. In the "Http Port" field, enter the Port used for the web config panel on the NVR

Playback

To use Hanwha in the playback form, the sdk must be installed. This can be found on dropbox at:

Technical\Deployment\4 Integrations\Hanwha

Known Issues

If an incorrect filepath is used when exporting clips, ExportManager will hang indefinitely on 0%. This is a bug with the SDK not realising that an invalid path has been given.
