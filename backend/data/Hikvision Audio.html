Overview

  • This guide will explain how to set up Hikvision audio within Viewcape and settings on the Hikvision that need to be changed
  • Audio to the NVR will not connect if a microphone isn't recognised by windows or if the NVR doesn't have a speaker connected
  • To listen to audio from cameras, make sure the stream type in the NVR is changed to Video/Audio, if Audio can't be heard from the webpage then it is unlikely it can be heard through viewscape

Setup

Viewscape

Audio Device

  • Go to Admin -> VTAS Pro -> Audio Device

  • Alias
    • The Name of the Hikvision Audio Device
  • IP Address
    • The IP Address of the NVR
  • Port
    • Audio port from the NVR -- Default 8000
  • Username
    • username to log into the NVR
  • Password
    • password to log into the NVR

Linking to Cameras

  • To Link an audio device to a camera, edit a cameras details and click on the Pro -> Audio Tab;

  • Click the Glasses icon and select the audio device
  • To listen to the audio from the camera, the camera must be linked to the NVR in the Admin -> VTAS Pro -> NVR section

Hikvision Setup

Audio from Workstation Microphone to NVR Speaker

  • Audio will not connect if a speaker isn't connected to the hik NVR
  • Audio will not connect if a microphone isn't connected to the workstation

Audio from Camera Microphone to Workstation via NVR

  • Microphone must be connected directly to camera
  • Video Type in Hikvision must be set to Video&Audio e.g.

  • Audio can be tested on the I've section of the webpage by selecting the camera, clicking on the speaker icon and changing volume:

Connecting to Audio

Audio from workstation microphone to Hikvision speaker

  • Click Pro -> Audio Announcer and select desired Audio Device

Or

  • Select a camera that is linked to the audio Device

  • Under the Pro Screen tab in Pro live, click the Microphone button in the bottom Left, this should make the Audio announcer appear in the top left of the screen with the audio device selected, click and hold 'Open Audio' to start the connection

Audio from Hikvision camera microphone to workstation speakers via NVR

  • Select the camera from the camera list
  • Under the Pro Screen tab in Pro live, click the speaker button in the top left. You should now be able to hear audio from the camera, click again to close audio

Support

  • There are currently no known issues with Quick Hit Timeouts, If you experience any issues, please contact your system integrator or Veracity Systems with as much info as possible

Errors

  • If the following error pops up please click Copy Error Details to Clipboard then Paste the details into an Email or a notepad to send to Veracity, If there is no way to copy the info from the machine then pressing "CTRL+S" will allow the error message to be seen and a photograph can be taken to be sent to support, if using this method please get a photo of the full message by scrolling down
