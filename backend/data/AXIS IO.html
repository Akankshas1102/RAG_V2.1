Overview

  • The AXIS Network IO Relay module is a piece of equipment used to connect to devices such as motion detectors or switches.

Functionality

  • The Axis device can connect to motion detectors or switches and send alerts from the device to Viewscape, which then can be integrated further.
  • You will need to make sure that this device connects to either a 12v power connection or a POE injector.

Pre-Requisites

  • You will need a web browser to access the web client which is used to change important settings for the device, such as the active state of the I/O or the type of I/O used or to add events (both discussed later in the guide).
  • You will need Viewscape installed so you can add in important events into Viewscape so you can see which doors are opening and closing as an example.
  • You will need Viewscape Settings Editor to change listening settings as well as what ports are being used for the AXIS device.
  • For testing you will need to connect "ground symbol" to a IO connection, once configured.

Configuring The Axis Device in Viewscape Settings Editor

  • To configure the Axis device, you will need to load up Viewscape Settings Editor on VS-VPS01 and you need to go to the Pro Server tab.

  • Then you will need to scroll down and look for the name "ListeningPortAxis" and assign it the Port number you gave it earlier within the Webpage for the module.

  • Then "save current page" within the Viewscape Settings Editor at the bottom to apply the changes.
