Viewscape 11.37.1 Manual

DM Alarms/Outputs

Setting up the DM unit (Net Vu SD Advanced)

Alarm -- Inputs

Aux 1 is the 25 pin connector on the back of the unit, Aux2 etc are expansion connectors

Contacts X and 25 need to have contact for an alarm to go off. If the ContactN/O is set to 1 then connecting contacts 1 and 25(ground) will set the alarm off

Zone Inputs

Set the Zone and Title (this will be used later to accept alarms to the server)

Individual Zones

Set the alarm zone as required e.g. 24hrs

Global Actions

These are the Pro Outputs

Network -- Remote Reporting

Enable Remote Reporting: Tick

Primary Hostname: Viewscape Server (IP Address)

Remote alarm reporting: Tick

Alarm Responder port: Must match ListeningPortDM in server settings

Testing

Go to System -- Status -- Alarm

Here you can see which alarms and inputs are active

On the back of the unit there is are 25 connectors for i/o alarms labelled alarms, connect alarms 1-24 with 25 (Ground to test the alarm
